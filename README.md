## Contribute
1. Fork the repository
2. Clone the repository & install required dependencies:
```
$ git clone git@github.com:YOUR_GITHUB_USERNAME/le-cash-bot-web.git
$ yarn install
$ cd client
$ yarn install
$ cd ..
```
3. Start the application:
```
$ cd client
$ yarn build
$ yarn start
```
or for Windows:
```
$ ./run
```
or for Linux:
```
$ sh run.sh
```
